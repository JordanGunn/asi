# Canonical minimal reference file set

Repositories that adopt this convention typically maintain a minimal reference file set:

```text
references/
├── 00_ROUTER.md
├── 01_SUMMARY.md
├── 02_CONTRACTS.md
├── 03_TRIGGERS.md
├── 04_NEVER.md
├── 05_ALWAYS.md
├── 06_PROCEDURE.md
└── 07_FAILURES.md
```
