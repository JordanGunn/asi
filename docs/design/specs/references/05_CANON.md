# Canonical Reference File Set (V2)

ASI V2 defines a smaller, deterministic reference canon. `00_ROUTER.md` is optional and used only for complex routing or idempotent lifecycle behavior.

```text
references/
├── 00_ROUTER.md           # optional
├── 01_SUMMARY.md
├── 02_INTENT.md
├── 03_POLICIES.md
└── 04_PROCEDURE.md
```
