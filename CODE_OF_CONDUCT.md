# Code of Conduct

This project is committed to a welcoming, harassment-free collaboration environment.

## Our standards

Expected behavior:

- Be respectful and professional.
- Assume good intent; ask clarifying questions.
- Prefer evidence and concrete examples over speculation.
- Keep discussions focused on the work and the spec.

Unacceptable behavior:

- Harassment, discrimination, or personal attacks.
- Doxxing or sharing private information without permission.
- Persistent disruption, trolling, or bad-faith engagement.

## Scope

This code of conduct applies to project spaces (issues, pull requests, discussions) and to direct communication about this project.

## Reporting

If you experience or witness unacceptable behavior, open an issue with the label `conduct` or contact the maintainers via a private channel if available.

## Enforcement

Project maintainers may remove, edit, or reject contributions that do not align with this code of conduct, and may temporarily or permanently ban contributors for unacceptable behavior.
