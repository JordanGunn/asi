# Always

- Always verify PLAN.md is approved before execution
- Always check for drift before execution
- Always check task dependencies before execution
- Always update task status to `in_progress` before starting
- Always update task status to `done` after completion
- Always report artifacts created or modified
- Always report execution errors immediately
- Always halt on error and await user direction
- Always produce execution receipt after each task
- Always append receipt to RECEIPT.md (create if not exists)
- Always log checkpoint after each file created or modified
- Always verify implementation matches PLAN.md specification
- Always trace every created file to a PLAN.md section
- Always log checkpoint before and after task execution
